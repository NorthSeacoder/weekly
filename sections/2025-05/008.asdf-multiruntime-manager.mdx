### [asdf - 一站式多运行时版本管理器](https://asdf-vm.com/zh-hans/)

![img](https://img.mengpeng.tech/i/2025/05/06/68195d5e84f5f.webp)

asdf 是一款强大的多运行时版本管理工具，支持通过单一命令行界面管理所有项目的语言环境和工具链。其插件生态丰富，几乎覆盖主流开发语言和工具，用户可通过简单 API 添加新工具支持。asdf 采用统一的 .tool-versions 配置文件，便于团队协作和环境复现，并兼容 nvm、rbenv 等主流配置。支持 Bash、ZSH、Fish 等多种 Shell，集成 GitHub Actions，方便在 CI/CD 流程中自动化环境搭建。适合需要跨项目、跨语言高效管理开发环境的开发者和团队。
