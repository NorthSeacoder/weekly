### [tweakcn：shadcn/ui 组件可视化主题编辑器](https://tweakcn.com)

![img](https://img.mengpeng.tech/i/2025/04/18/6801ebabaf062.webp)

tweakcn 是一款专为 shadcn/ui 组件和 Tailwind CSS 设计的可视化无代码主题编辑器，帮助开发者和设计师轻松打造独特、个性化的 UI 风格，提升前端开发效率。

- 提供丰富的主题预设和可视化调色板，支持一键切换和实时预览，极大简化 UI 个性化流程
- 支持对 shadcn/ui 组件的深度定制，未来将覆盖更多组件类型，满足多样化项目需求
- 无需编写代码，所见即所得，适合前端开发者、设计师及产品经理协作使用
- 开源免费，社区活跃，持续更新，支持本地部署与二次开发
