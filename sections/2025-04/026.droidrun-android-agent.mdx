### [DroidRun：LLM 智能体驱动的 Android 设备自动化框架](https://github.com/droidrun/droidrun)

![img](https://img.mengpeng.tech/i/2025/04/18/6801e8725542e.webp)

DroidRun 是一款开源的 Android 设备自动化控制框架，结合大语言模型（LLM）智能体，实现通过自然语言指令远程操控 Android 设备：

- 支持 OpenAI、Anthropic、Gemini 等多家主流 LLM 提供商，CLI 简洁易用，Python API 便于二次开发
- 通过 ADB 与 DroidRun Portal App 实现设备连接，支持本地与云端多场景部署
- 可自动分析设备截图，实现视觉理解与 UI 元素识别，适合自动化测试、远程协助、批量操作等场景
- 支持多模型切换、任务步骤限制、设备多选等高级功能，满足复杂自动化需求
- 社区活跃，文档完善，MIT 许可，适合开发者和自动化爱好者快速上手与扩展
