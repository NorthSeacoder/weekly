### [GitMCP：开源远程 MCP 服务器，助力 AI 智能理解代码](https://github.com/idosal/git-mcp)

![img](https://img.mengpeng.tech/i/2025/04/18/6801ea675c25b.webp)

GitMCP 是一款免费开源的远程 MCP 服务器，专为 AI 助手与 GitHub 项目无缝集成设计，显著提升 AI 对代码和文档的理解能力，减少"幻觉"现象。

- 支持多种 AI 助手（如 Cursor、Claude、VSCode 等）通过标准协议访问任意 GitHub 仓库，获取文档、代码和上下文信息
- 提供 fetch/search 等丰富工具，支持智能检索、代码搜索、外链内容抓取等多种场景
- 部署简单，支持自托管和云端使用，兼容所有公开 GitHub 项目，隐私安全有保障
- 开源免费，社区活跃，持续更新，适合开发者、AI 产品团队和开源爱好者扩展应用
