### [Fumadocs：美观高效的开源文档框架](https://fumadocs.vercel.app/)

![img](https://img.mengpeng.tech/i/2025/04/18/6801eb5a14df2.webp)

Fumadocs 是一款专为开发者打造的开源文档框架，基于 Next.js，支持 MDX、内容集合等多种内容源，帮助团队和个人快速构建美观、灵活且高性能的文档站点。

- 一键初始化项目，支持 MDX、内容集合、OpenAPI 等多种内容格式，极大简化文档搭建流程
- 内置丰富主题与组件，支持自定义 UI、深色模式、Shiki 代码高亮，兼容 React Server Component
- 集成 Orama/Algolia 全文搜索，支持自动化内容处理、类型安全校验与 OpenAPI 文档生成
- 轻量高效，专为开发者体验和可维护性设计，开源免费，社区活跃，适合个人与团队长期使用
