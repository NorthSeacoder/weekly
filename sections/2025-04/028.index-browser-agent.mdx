### [Index：AI 驱动的开源浏览器自动化智能体](https://github.com/lmnr-ai/index)

![img](https://img.mengpeng.tech/i/2025/04/18/6801e96d67798.webp)

Index 是一款开源的浏览器自动化智能体，用户无需编写脚本，仅用自然语言即可自动完成复杂网页操作，极大降低自动化门槛，提升网页任务处理效率。

- 基于 Claude 3.7 Sonnet 等大模型，支持 API 和可视化 UI，适合数据采集、表单填写、内容分析等多种场景
- 支持自托管和远程 CDP 连接，部署灵活，满足本地与云端多样化需求
- 无需代码，任务描述简单直观，开发者和自动化需求者均可高效上手
- 开源免费，社区活跃，持续更新，适合个人和团队扩展应用
