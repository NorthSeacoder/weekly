### [AI Use 自动化工具合集：让 AI 控制万物](https://twitter-thread.com/t/1911955081630818574)

![img](https://img.mengpeng.tech/i/2025/04/18/6801eabfcc475.webp)

AI Use 自动化工具合集汇集了多款开源 AI 自动化工具，覆盖浏览器、桌面、手机等多场景，助力开发者和 AI 爱好者实现自然语言驱动的全平台自动化操作。

- 浏览器自动化：如 Steel Browser、mcp-browser-use，支持 AI 代理自动化 Web 操作与 Web 研究，无需关心底层基础设施
- 桌面自动化：如 Surf、E2B Desktop Sandbox，结合虚拟桌面与 AI API，安全高效地实现桌面级自动化
- 手机自动化：如 Droidrun，支持 AI 代理与 Android 应用无缝交互，适合移动端自动化需求
- MCP 生态：如 GitHub MCP Server，提供与 GitHub API 的无缝集成，支持高级自动化和交互
