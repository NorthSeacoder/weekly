### [Magic UI - 动画UI组件库](https://magicui.design/)

![img](https://img.mengpeng.tech/i/2024/12/05/675136e198511.webp)

来源: [magicui.design](https://magicui.design/)

Magic UI 是一个专注于动画效果的 React UI 组件库，提供 50+ 免费开源的动画组件，完美配合 shadcn/ui 使用。

#### 核心特性
- 丰富组件：50+ 免费开源动画组件
- 技术栈：React + TypeScript + Tailwind CSS
- 动画支持：集成 Framer Motion 动画
- 类型安全：完整的 TypeScript 类型支持
- 高度定制：支持自定义主题和样式

#### 应用案例
- YC 公司：多家 YC 孵化的创业公司在用
- 知名项目：被 Google Ventures 投资的项目采用
- 社区推荐：受到 t3.gg 等技术社区推荐
- 商业应用：多个商业项目成功案例
