### [Prompt Engineering 学习资源大全](https://twitter-thread.com/t/1830487029139800266)

这是一个精心整理的 Prompt Engineering 学习资源合集，包含了从入门到进阶的各类学习材料。

#### 基础教程
- [Prompt Engineering Guide](https://www.promptingguide.ai/)：最全面的开源提示工程指南，包含理论基础、实践技巧和高级应用，支持中文等多语言
- [Anthropic Prompt Engineering](https://docs.anthropic.com/claude/docs/prompt-engineering)：Claude 官方提供的提示工程最佳实践，包含系统提示、角色设定和任务分解等进阶技巧
- [OpenAI Prompt Engineering](https://platform.openai.com/docs/guides/prompt-engineering)：ChatGPT 官方提示工程指南，涵盖六大原则和常见应用场景的详细示例

#### 进阶资源
- [Anthropic Prompt Library](https://docs.anthropic.com/claude/docs/prompt-library)：Claude 官方提供的提示词模板库，包含 40+ 个经过验证的高质量提示词模板
- [Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook)：Claude API 的官方示例集，包含代码实现和最佳实践，适合开发者参考
- [OpenAI Cookbook](https://cookbook.openai.com/)：OpenAI 官方的 API 使用指南，包含 100+ 个实用代码示例和完整项目案例
- [OpenAI Examples](https://platform.openai.com/examples)：ChatGPT 官方提供的提示词示例库，包含 80+ 个经过优化的任务型提示词

#### 视频课程
- [ChatGPT Prompt Engineering for Developers](https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/)：吴恩达与 OpenAI 合作推出的提示工程课程，面向开发者的实战指南
- [Generative AI for Everyone](https://www.deeplearning.ai/courses/generative-ai-for-everyone/)：面向所有人的生成式 AI 入门课程，包含提示工程的基础知识和实践技巧
- [Stanford CS25](https://www.youtube.com/watch?v=P127jhj-8-Y&list=PLoROMvodv4rNiJRchCzutFw5ItR_Z27CM)：斯坦福大学的 Transformers 课程，深入讲解大语言模型的原理和应用
- [IBM Technology](https://www.youtube.com/@IBMTechnology)：IBM 官方的 AI 技术频道，定期更新提示工程和 AI 应用的最新进展

#### 实践平台
- [Anthropic Prompt Generator](https://docs.anthropic.com/claude/docs/prompt-engineering/prompt-generator)：基于 Claude 的智能提示词生成工具，可根据需求自动生成和优化提示词
- [Microsoft AI Course](https://github.com/microsoft/generative-ai-for-beginners/)：微软官方的 AI 入门教程，包含 12 节课程和完整的动手实践项目
- [Google AI Resources](https://developers.google.com/machine-learning/resources/prompt-eng)：Google 提供的 AI 学习资源，包含 PaLM API 的提示工程最佳实践和示例

这些资源涵盖了从理论到实践的各个方面，适合不同层次的学习者。建议先从基础教程开始系统学习，再通过实践平台积累经验，最后深入研究进阶资源。
