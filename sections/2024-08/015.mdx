### [Product Hunt 每日中文热榜](https://github.com/ViggoZ/producthunt-daily-hot)

![img](https://img.mengpeng.tech/i/2024/11/24/6742f9c32d54a.webp)

来源: [github.com/ViggoZ/producthunt-daily-hot](https://github.com/ViggoZ/producthunt-daily-hot)

这是一个基于 GitHub Actions 和 GPT-4 的自动化工具，每天定时抓取 Product Hunt 热门产品，并生成中文榜单，帮助中文用户发现优质产品。

#### 核心功能
- 自动数据获取：每天抓取 Product Hunt Top 30 产品信息
- AI 翻译优化：使用 GPT-4 进行高质量的产品描述翻译
- 智能标签：自动生成中文关键词，便于产品分类和理解
- 自动化发布：支持自动发布到 WordPress 站点
- 定时更新：每天下午4点自动更新榜单内容

#### 技术特点
- GitHub Actions：利用工作流实现自动化部署
- OpenAI 集成：使用 GPT-4 进行智能翻译和关键词提取
- Markdown 生成：自动生成结构化的产品信息文档
- WordPress API：支持自动发布到 WordPress 平台
- 配置灵活：支持自定义更新时间和内容格式

#### 使用场景
- 产品经理：及时了解全球最新产品动态
- 创业者：获取产品灵感和市场趋势
- 开发者：学习优秀产品的设计思路
- 内容创作者：获取高质量的产品资讯素材

这是一个非常实用的工具，通过自动化和 AI 技术，帮助中文用户跨越语言障碍，更便捷地获取全球最新的产品资讯。
