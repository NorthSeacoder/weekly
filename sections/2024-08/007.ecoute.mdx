### [Ecoute](https://github.com/SevaSk/ecoute)

![img](https://img.mengpeng.tech/i/2024/12/09/675683fa1a691.webp)

来源: [github.com/SevaSk/ecoute](https://github.com/SevaSk/ecoute)

Ecoute 是一个实时语音转录工具，可以同时转录用户的麦克风输入和扬声器输出，并使用 GPT-3.5 为用户生成建议回复，帮助用户更好地进行对话。

#### 核心功能
- 实时转录:同时支持麦克风输入和扬声器输出的实时语音转录
- AI 辅助:使用 OpenAI GPT-3.5 根据对话内容生成合适的回复建议
- 双模式支持:提供本地 Whisper 模型和 Whisper API 两种转录模式
- 多语言支持:使用 API 模式时支持多种语言的转录

#### 技术特点
- 语音识别:使用 OpenAI 的 Whisper 模型进行语音识别
- 低资源消耗:默认使用 Whisper 'tiny' 模型以确保快速响应
- 系统集成:支持系统默认音频设备的输入输出
- 开发框架:基于 Python 实现，支持 Windows 系统

#### 使用场景
- 在线会议:实时转录会议内容，辅助理解和记录
- 语言学习:通过实时转录和 AI 建议提升口语能力
- 客服工作:协助客服人员更好地回应客户问题
- 访谈记录:自动记录访谈内容，减少人工记录工作

这是一个非常实用的开源工具，特别适合需要语音转录和对话辅助的场景，可以显著提升沟通效率和质量。
