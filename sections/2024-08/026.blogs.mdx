#### [国内博客](https://github.com/timqian/chinese-independent-blogs)

来源: [@YanHuiMi](https://x.com/YanHuiMi/status/1822408538192543920)

![img](/sections/2024-08/026.inner.png)  

Chinese Independent Blogs 是一个 GitHub 项目，汇集了中文独立博客的列表。该项目旨在推广中文独立博客文化，提供一个平台让更多人发现和阅读这些博客。项目包含一个 OPML 文件，方便用户导入到自己的 RSS 阅读器中。
