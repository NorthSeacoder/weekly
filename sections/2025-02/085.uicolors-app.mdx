### [UI Colors - Tailwind CSS 颜色生成器 & 编辑工具](https://uicolors.app/create)

![img](https://img.mengpeng.tech/i/2025/02/28/67c14c6ca5f39.webp)

[UI Colors](https://uicolors.app/create) 是一款 Tailwind CSS 颜色生成器和编辑工具，旨在帮助开发者快速创建自定义的颜色方案。用户可以通过空格键、输入 HEX 代码或调整 HSL 或 OKLCH 值来生成自定义颜色阶梯。 它允许用户调整 primary, secondary 和 tertiary 输入颜色，并实时预览 UI 示例效果。 UI Colors 专注于为 Tailwind CSS 提供便捷的颜色方案生成和管理，提高开发效率。
