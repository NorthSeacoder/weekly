![img](https://img.mengpeng.tech/i/2025/02/17/67b2cf210d6de.webp)

本文介绍了如何在家里的 NAS 或软路由上部署 Open WebUI，并利用其免费白嫖大厂 DeepSeek R1 满血 API。  Open WebUI 具有以下特点:Docker 一键部署、非常稳定快速、支持联网搜索、支持直接预览 Artifacts、支持自定义设置（如温度）以及开源免费且更新迅速。  用户可以在家里的手机、电脑、平板等设备上使用 Open WebUI，享受 DeepSeek R1 API 的强大功能。  对于拥有 NAS 或软路由，并且希望免费体验 DeepSeek R1 API 的用户来说，Open WebUI 是一个不错的选择。  项目开源地址为 [github.com/open-webui/open-webui](https://github.com/open-webui/open-webui)。
