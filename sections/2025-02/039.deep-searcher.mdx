### [DeepSearcher](https://github.com/zilliztech/deep-searcher)

![img](https://img.mengpeng.tech/i/2025/02/17/67b2c35257f49.webp)

DeepSearcher 是一个开源项目，旨在结合强大的 LLMs (如 DeepSeek, OpenAI) 和向量数据库 (如 Milvus) ，实现基于私有数据的**深度搜索、评估和推理**。 它能够为用户提供高度准确的答案和全面的报告，适用于企业知识管理、智能问答系统和信息检索等多种场景。 DeepSearcher 的主要特点包括：**私有数据搜索** (可结合在线内容提升准确性)；**向量数据库管理** (支持 Milvus 等，可数据分区)；**灵活的 Embedding 选项** (兼容多种模型)；**多 LLM 支持** (DeepSeek, OpenAI 等)；以及**文档加载器** (支持本地文件，Web 爬虫开发中)。 项目提供了快速上手的 Demo 和详细的配置说明，并计划在未来增强 Web 爬虫功能、支持更多向量数据库和 LLM，以及提供 RESTful API 接口。 对于希望构建基于私有数据的智能搜索和问答系统的开发者，DeepSearcher 是一个值得关注的开源方案。
