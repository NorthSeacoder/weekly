### [deepseek-ai/awesome-deepseek-integration](https://github.com/deepseek-ai/awesome-deepseek-integration)

![img](https://img.mengpeng.tech/i/2025/02/10/67a9d800902f1.webp)

Awesome DeepSeek Integration 是一个精选列表，汇总了 DeepSeek API 在各种软件中的集成应用。DeepSeek 开放平台提供了 API 密钥，方便开发者将 DeepSeek 的强大功能融入到各种应用中。

**主要集成类别包括:**

-   **应用程序:** 涵盖桌面客户端 Chatbox, 跨平台 Web UI ChatGPT-Next-Web, 微信助手 Liubai, iOS 聊天客户端 Pal, 开源应用 LibreChat, Just-Chat, Zotero 插件 PapersGPT, RSS 翻译器 RSS Translator, AI 入口 Enconvo, 桌面 AI 助手 Cherry Studio, iOS 应用 ToMemo, 视频字幕工具 Video Subtitle Master, Webapp Chatworm, macOS 截图 OCR 工具 Bob, Neovim 插件 codecompanion.nvim 等。
-   **JetBrains 扩展:** 包括 AutoDev, Onegai Copilot, Continue, 中英翻译插件 Chinese-English Translate, AI Git Commit 等，为 JetBrains IDE 提供 AI 辅助功能。
-   **Discord 机器人:** Geneplore AI 是运行大型 AI Discord 机器人的代表，集成了 Deepseek v3 和 R1 模型。
-   **编辑器:** Cursor 作为 AI 代码编辑器被收录。
-   **Emacs:** gptel 和 Minuet AI 为 Emacs 用户提供 LLM 客户端。
-   **其他:** ShellOracle (终端命令生成工具), Bolna (对话式语音 AI 代理), siri_deepseek_shortcut (Siri 快捷指令), n8n-nodes-deepseek (N8N 工作流节点), Portkey AI (统一 API 网关), LiteLLM (Python SDK), Mem0 (AI 助手记忆层), promptfoo (LLM Prompt 测试工具), deepseek-tokenizer (分词库), Langfuse (LLM 可观测性平台), deepseek-review (代码审查工具), GPTLocalost (Word 本地使用 DeepSeek-R1), WordPress ai 助手 (WordPress AI 助手插件) 等工具和服务。
