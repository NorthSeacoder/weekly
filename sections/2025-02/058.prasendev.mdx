### [Prasendev - 基于 Next.js 14 的现代响应式作品集网站模板](https://github.com/StarKnightt/prasendev)

![img](https://img.mengpeng.tech/i/2025/02/27/67bfef789a558.webp)

Prasendev 是一个使用 Next.js 14、TypeScript 和 Tailwind CSS 构建的现代作品集网站模板。它具有响应式设计，支持深色/浅色模式，并集成了博客功能、视频画廊和产品推荐列表。该模板还利用 Framer Motion 实现流畅的动画效果，并通过 GitHub 集成展示实时的贡献图。Prasendev 注重 SEO 优化和性能，是一个快速启动个人作品集网站的优秀选择。
