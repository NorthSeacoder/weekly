### [歸藏(guizang.ai) - Windsurf MCP 教程:扩展 Copilot 能力，连接外部数据源](https://twitter-thread.com/t/1892056272381542405)

![img](https://img.mengpeng.tech/i/2025/02/27/67bffb5b47b3b.webp)

[歸藏(guizang.ai) (@op7418) 在 Twitter 上分享](https://twitter-thread.com/t/1892056272381542405) 了 Windsurf 出品的 MCP (Model Context Provider) 视频教程。MCP 的核心价值在于为新一代 Agent (如 Cascade) 提供标准化接口，安全地访问外部数据源，例如 Slack、GitHub 和 Google Drive 等。 传统 Copilot 只能基于代码库回答问题，而 MCP 扩展了 Copilot 的能力，使其能够连接更多数据源，将数据转化为可操作的上下文。 教程通过集成 Slack 频道的实战案例，展示了如何使用 MCP 让 Cascade 访问 Slack 中的编码规范讨论，并指导代码重构，显著提升了代码生成质量和场景适用性。
