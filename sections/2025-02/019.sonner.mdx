### [sonner：React 平台的精致 Toast 组件](https://github.com/emilkowalski/sonner)

![img](https://img.mengpeng.tech/i/2025/02/10/67a9df0ad1dd4.webp)

Sonner 是一个为 React 应用设计的精致且注重用户体验的 Toast (通知) 组件。它以其简洁的 API 和优雅的设计，帮助开发者轻松地在 React 应用中集成美观的通知提示。Sonner 提供了开箱即用的样式，同时也支持高度自定义，以满足不同项目的 UI 风格需求。

**主要特点:**

-   **React 组件:** 专为 React 应用构建，易于集成和使用。
-   **简洁 API:** 提供 `toast()` 函数，从应用的任何位置轻松触发通知。
-   **开箱即用:** 默认样式美观，无需额外配置即可快速上手。
-   **高度可定制:** 支持自定义 Toast 的样式、持续时间、位置等。
-   **主题化:** 支持亮色和暗色主题。
-   **无障碍:** 注重无障碍设计，提升用户体验。
-   **轻量级:** 体积小巧，不会增加应用负担。
