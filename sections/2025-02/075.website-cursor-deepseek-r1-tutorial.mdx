### [使用 Cursor 快速创建官网 + DeepSeek R1 智能客服实战教程](https://twitter-thread.com/t/1892786136189456797)

![img](https://img.mengpeng.tech/i/2025/02/27/67bffd30b1b76.webp)

[熠辉 Indie (@yihui_indie) 在 Twitter 上分享](https://twitter-thread.com/t/1892786136189456797) 了最新的实战课程，主题是 "用 Cursor 快速创建官网 + 添加满血版 DeepSeek R1 智能客服"。 本课程旨在教授用户如何使用开源模板和 Cursor 的 Agent 模式快速修改创建网站，并接入火山引擎的满血版 DeepSeek-R1 模型，搭建 Dify + DeepSeek-R1 智能客服系统，最终将智能客服集成到自己的网站中。 课程内容涵盖了网站快速搭建、智能客服创建和集成等多个实用技能，并提供配套视频和详细文字教程，方便用户学习。 课程中使用的开源模板来自 template0 的 shadcn-landing-page。
