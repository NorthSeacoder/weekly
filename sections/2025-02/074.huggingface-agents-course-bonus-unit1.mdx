### [Hugging Face Agents Course Bonus Unit 1 - 微调 LLM 以实现 Function Calling](https://huggingface.co/learn/agents-course/bonus-unit1/introduction)

![img](https://img.mengpeng.tech/i/2025/02/27/67bffc8b2097e.webp)

Hugging Face Agents Course 的 Bonus Unit 1 介绍了如何微调大型语言模型 (LLM) 以实现 Function Calling 技术。Function Calling 是一种重要的 LLM 技术，它训练模型在训练阶段采取行动并解释观察结果，从而增强 AI 的鲁棒性。本单元将讲解 Function Calling 的核心概念，并介绍 LoRA 高效微调方法，以及 Thought → Act → Observe 循环在 Function Calling 模型中的应用。通过学习本单元，你将能够理解 Function Calling 的内部工作原理，并掌握微调模型以实现 Function Calling 的技能。
