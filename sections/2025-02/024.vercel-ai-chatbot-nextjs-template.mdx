### [vercel/ai-chatbot](https://github.com/vercel/ai-chatbot)

![img](https://img.mengpeng.tech/i/2025/02/14/67aecd76198b5.webp)

Vercel AI Chatbot 是一个使用 Next.js 14 和 AI SDK 构建的全功能开源聊天机器人模板。它利用 Next.js App Router 的高级路由和 React Server Components，提供流畅的导航和高性能。AI SDK 统一了与多种 LLM 提供商（如 OpenAI、Anthropic、Cohere 等）的接口，简化了文本生成、结构化对象和工具调用的处理。模板集成了 shadcn/ui 组件库，使用 Tailwind CSS 进行样式设计，并采用 Radix UI 的组件基元以保证可访问性和灵活性。数据持久化方面，利用 Vercel Postgres 和 Vercel Blob 实现聊天记录和用户数据的存储。同时，NextAuth.js 提供了简单而安全的身份验证方案。默认使用 OpenAI `gpt-4o` 模型，但可以轻松切换到其他模型提供商。一键部署到 Vercel，或本地运行，是快速搭建和定制 AI 聊天机器人的理想选择。
