### [jiayuan 发布 Devv Builder (Beta)](https://devv.ai/zh/build)

![img](https://img.mengpeng.tech/i/2025/02/14/67aed4bf3e302.webp)

Devv Builder (Beta) 是一款 AI 后端代码生成工具，可以根据用户需求自动生成后端代码并部署为可用的 API 服务。用户只需输入需求，Devv Builder 即可完成后端代码的生成、Supabase 表结构的一键创建/修改、API 服务部署等一系列操作。它内置 Playground，方便用户进行 API 测试，并能自动生成 API 文档，方便前端或客户端开发者进行后续开发。Devv Builder 目前支持 TypeScript，未来计划支持更多语言和框架，并扩展多云部署、定时任务、数据管道、GitHub 集成和团队协作等功能。用户可以在 [hub.devv.ai](http://hub.devv.ai) 提交功能需求。
