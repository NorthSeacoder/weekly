### [Languine Monorepo Cursor 项目规则 (.mdc) 配置探讨](https://twitter-thread.com/t/1891605846821568920)

![img](https://img.mengpeng.tech/i/2025/02/27/67bff062ecb99.webp)

该 Twitter 帖子由 Pontus Abrahamsson 分享，讨论了 Languine monorepo 项目中用于 Cursor 编辑器的规则配置文件（`.mdc` 文件）。该项目采用了 Next.js, tRPC, Supabase 和 React Email 等技术栈。帖子内容着重介绍了如何使用 `.mdc` 文件来配置 Cursor 编辑器的项目规则，为开发者提供了在类似技术栈的 Monorepo 项目中进行 Cursor 规则配置的参考。
