### [Obsidian Web Clipper](https://obsidian.md/clipper)

![img](https://img.mengpeng.tech/i/2025/02/18/67b40f670c53d.webp)

Obsidian Web Clipper 是一款免费开源的浏览器插件，旨在帮助用户轻松将网页内容保存到 Obsidian 知识库中。  用户可以通过一键点击，快速捕获网页文章、食谱、参考文献、学术论文等多种类型的内容，并支持自定义模板，灵活控制保存格式和元数据。  Web Clipper 可以高亮网页内容，方便用户选择需要保存的文本、图片和区块。  所有剪辑的内容都以 Markdown 格式存储在本地 Obsidian Vault 中，确保数据隐私和持久性，并支持导出为 JSON 格式，避免厂商锁定。  此外，Web Clipper 提供快捷键操作和自动应用模板等高级功能，提升内容剪辑效率。  这款插件适用于 Chrome, Safari, Firefox, Edge, Brave, Arc, Orion 和 Vivaldi 等主流浏览器，是 Obsidian 用户构建个人知识库的必备工具。
