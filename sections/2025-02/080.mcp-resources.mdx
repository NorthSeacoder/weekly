### [MCP (Model Context Provider) 优秀资源列表与工具推荐](https://twitter-thread.com/t/1893336389116203256)

![img](https://img.mengpeng.tech/i/2025/02/28/67c14a2dde9b8.webp)
向阳乔木 (@vista8) 在 Twitter 上分享了两个 MCP (Model Context Provider) 的优秀资源。 MCP 协议旨在为大模型和本地客户端提供一种 API 接口，使 LLM 能够自动识别和调用外部工具及数据源。 帖子中推荐了 [Awesome MCP Server List](https://github.com/appcypher/awesome-mcp-servers) GitHub 仓库，以及 [Smithery](https://smithery.ai/) 网站，该网站专门收录 MCP Server，并支持 Claude 客户端、Cursor、Windsurf、Cline 等工具的一键复制安装命令。 通过 MCP，AI 工具可以连接更多外部数据和功能，例如 Brave 浏览器的搜索 API 和 fetch URL to Markdown 服务，从而扩展 AI 助手的能力，实现更便捷的任务自动化。
