### [Hass-Panel - 基于 React 开发的 Home Assistant 控制面板](https://github.com/mrtian2016/hass-panel)

![img](https://img.mengpeng.tech/i/2025/02/28/67c14b55e6937.webp)

[Hass-Panel](https://github.com/mrtian2016/hass-panel) 是一个基于 React 开发的开源智能家居控制面板，旨在简化 Home Assistant 的使用体验。它可以通过 Docker 或 Home Assistant Addon 方式一键部署，无需编程经验，开箱即用。Hass-Panel 具有响应式设计，支持移动端和桌面端，并提供高度可配置的拖拽布局、PWA 应用支持和美观的用户界面。 它集成了丰富的设备支持，包括灯光、空调、摄像头、传感器等，并通过 JWT 认证和密码加密存储确保安全。 Hass-Panel 致力于打造最简单的 Home Assistant 控制面板，让用户可以轻松管理和监控智能家居设备。
