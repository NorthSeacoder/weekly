### [Rosen Charts - 面向未来的图表库](https://rosencharts.com/)

![img](https://img.mengpeng.tech/i/2025/03/07/67ca7d2a9e526.webp)

-   Rosen Charts 是首个完全支持 React Server Components (RSC) 的图表库，可显著减少客户端 bundle 大小
-   提供即复制即用的图表组件，支持从简单的条形图到复杂的散点图等多种图表类型
-   完全可定制，支持通过直接访问 div 和 svg 来自定义每个细节
-   原生支持暗色模式，并与 Tailwind、Shadcn、NextJS、Remix 等现代技术栈完美集成
-   基于 HTML 和 D3 构建，确保了跨框架的兼容性和可移植性
