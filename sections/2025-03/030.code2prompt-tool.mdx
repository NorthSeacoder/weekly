### [code2prompt：智能代码提示词生成工具](https://github.com/mufeedvh/code2prompt)

![img](https://img.mengpeng.tech/i/2025/03/08/67cc46234a700.webp)

-   强大的命令行工具，可将整个代码库转换为结构化的 LLM 提示词，支持源码树结构和令牌计数
-   提供丰富的模板系统和自定义变量，适用于代码文档生成、安全审计、性能优化等多种场景
-   支持多种主流编码器（cl100k、p50k 等），并提供 Python SDK 方便集成到现有工作流程中
