### [OpenManus：让 AI Agent 开发更简单](https://github.com/mannaandpoem/OpenManus)

![img](https://img.mengpeng.tech/i/2025/03/08/67cc44c8c089a.webp)

-   由 MetaGPT 团队成员开发的开源 AI Agent 项目，提供类似 Manus 的功能但无需邀请码
-   采用简洁的实现方式，支持多种 LLM 模型，可快速部署和定制个性化 Agent
-   活跃的社区支持和清晰的发展路线图，包括规划能力增强、标准化评估和容器化部署等
